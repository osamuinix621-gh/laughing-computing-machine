## 1. AlternativeTo

>[AlternativeTo - Crowdsourced software recommendations](https://alternativeto.net/)

「〇〇と似ているアプリケーション」を「OSを問わず」検索できるサイトです。


<details><summary>くわしく</summary>

- 	英語サイト
	-	日本語サイトなし
	- 	翻訳によっては名称まで変わってしまう
- 	各アプリケーションの公式リンクは`More About 〇〇`から進んで個別ページの右下にあります
- 	個別ページの`〇〇 alternatives`から似たアプリをさらに検索可能
- 	LinuxやWeb版も掲載あり
-  	ゲームも検索可能
- 	ダウンロードにはご注意ください

</details>



## 2. SteamPeek

>[SteamPeek - Indie friendly game discovery](https://steampeek.hu/)

Steam内の「〇〇と似ているゲーム」を検索できるサイトです。

<details><summary>くわしく</summary>

- 英語サイト
	- 日本語ページなし
 	- 翻訳によっては名称まで変わってしまう
- ゲーム配信プラットフォームSteamで配信しているゲーム専用
- 検索キーワードは存在するゲーム名にしないと検索できない仕様
	- サジェストが出るので合致するものを拾えれば大丈夫です
- 色々なゲームがあるので（意味深）18+ 対象フィルタを確認するメッセージが出ます
- アクセスが遅い場合もあります

</details>

## 4. PopClip
 
>[PopClip for Mac](https://www.popclip.app/)

テキストを選択したときに、iOS風のポップアップを出します。


<details><summary>くわしく</summary>

- 有料
	- 無料期間あり
- 日本語表記対応
- 選択したテキストに対してiOS風のポップアップでアクションを実行
  - コピー・ペーストのほか、全角化・小文字化・URLをChromeで開くetc...
- アクションは公式サイトから追加可能

</details>





## 5. BetterTouchTool

>[BetterTouchTool folivora.ai - Great Tools for your Mac!](https://folivora.ai/)

Mac内蔵トラックパッド・Magic Trackpad・Magic Mouse・キーボードの操作（ジェスチャ）などをトリガーとしてアクションを実行します。

<details><summary>くわしく</summary>

- 有料
	- 無料期間あり
	- 2年ライセンス / 永続ライセンス / サブスクリプション から選択
		- サブスクリプションは別の運営
- 日本語表示は自動翻訳




</details>

<details><summary>たとえば</summary><br>

| デバイス・条件 | トリガー | アクション |
|---|---|---|
| Magic Mouse / Trackpad | 3本指クリック | ⌘ V |
| Magic Mouse  | 4本指クリック | ⌘ W |
| Finder のときだけ | 3本指クリック | ⌘ Click |
| MacBookトラックパッド | 4本指スワイプ | アプリ A・B を終了、C・D・E を起動 |
| キーボード | テキストエリアに `B` `T` `T` `L` と順に入力 | `BetterTouchTool` に置き換え |
| カーソル | 画面左端に触れる | 新しいウインドウを開く |
<br>
</details>

<details><summary>個人的には</summary><br>


似たことをできるアプリケーションは他にも存在します。

無料公開の頃から気に入っていたので購入しましたが、これを最初に導入するのは有料の敷居が高いかもしれません。

メニューバートリガー用アイコンを置いたり、フローティングウインドウにショートカットボタンをパレット表示なども。

日本語表示はできますが、自動翻訳のわかりづらさと解説サイトに調べるときに対照が面倒なので、英語表示にしています。

トラックパッド / Magic Trackpad 向けの紹介記事が検索で多く見つかります。

個人的にはMagic Mouse の使い勝手を上げることに重点を置いて使っています。

アプリケーション内で英語に切り替えられないものでも`システム設定`で設定することもできます

</details>
