# laughing-computing-machine

## パソコンのアプリケーション紹介





### 1. AlternativeTo

>[AlternativeTo - Crowdsourced software recommendations](https://alternativeto.net/)



「〇〇と似ているアプリケーション」を検索できるサイトです。

<details><summary>1.1 くわしく</summary>
- 英語サイト（日本語ページなし）
	- 翻訳によっては名称まで変わってしまうのでご注意ください
	- 各アプリケーションの公式リンクは個別ページの右下にあります
- LinuxやWebツールの掲載あり
- OS・価格などでフィルタ可能
- ダウンロードにはご注意ください
</details>

<details><summary>1.2 ちなみに</summary>

- ゲームも検索可能
</details>


--

### 2. SteamPeek

>[SteamPeek - Indie friendly game discovery](https://steampeek.hu/)

Steam内の「〇〇と似ているゲーム」を検索できるサイトです。

<details><summary>2.1 くわしく</summary>
- 英語サイト（日本語ページなし）
- Steamで配信しているゲーム専用
 - 検索キーワードは存在するゲーム名にしないと検索できない仕様
- 色々なゲームがあるので（意味深）18+ 対象フィルタを確認するメッセージが出ます
- Steamの画像を読み込んでいるためか、アクセスはSteamサイトくらいの重さを感じます
</details>

--

### 3. BetterTouchTool

>[BetterTouchTool folivora.ai - Great Tools for your Mac!](https://folivora.ai/)

Macのトラックパッドなどでできる操作を設定します。システム環境設定よりも多く。

<details><summary>3.1 くわしく</summary>
- Mac内蔵トラックパッド・Magic Trackpad・Magic Mouse・キーボードなどで決められたトリガーに対してアクションを実行します
- 日本語表示は自動翻訳
- 有料（無料期間あり）
	- 2年ライセンス / 永続ライセンス / サブスクリプション から選択
		- サブスクリプションは別の運営


</details>

<details><summary>3.2 たとえば</summary>

| デバイス・条件 | トリガー | アクション |
|---|---|---|
| Magic Mouse / Trackpad | 3本指クリック | ⌘ V |
| Magic Mouse  | 4本指クリック | ⌘ W |
| Finder のときだけ | 3本指クリック | ⌘ Click |
| MacBookトラックパッド | 4本指スワイプ | アプリ A・B を終了、C・D・E を起動 |
| キーボード | `B` `T` `T` `L` と順に入力 | `BetterTouchTool` に置き換え |
</details>

<details><summary>3.3 ちなみに</summary>

- 無料公開期間から気に入っていたので購入しましたが、導入には有料の敷居が高いかもしれません
- 日本語の自動翻訳がわかりづらいため、個人的にはシステム環境設定でアプリの言語を英語に設定しています
- リリースノートを読んでいないと、知らないうちに機能が追加されていることがあります
- 複数のアクションをまとめて実行することも可能
- トラックパッド / Magic Trackpad 向けの紹介が多いですが、個人的にはMagic Mouse の使い勝手を上げることに重点を置いて使っています
</details>

### 4. PopClip
 
>[PopClip for Mac](https://www.popclip.app/)

テキストを選択したときに、iOS風のポップアップを出します。


<details><summary>4.1 くわしく</summary>

- 有料（無料期間あり）
- 選択したテキストに対してiOS風のポップアップでアクションを実行
  - コピー・ペーストのほか、全角化・小文字化・URLをChromeで開く、など
- 拡張機能は公式サイトから追加可能
</details>
